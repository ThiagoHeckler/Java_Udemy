## Sobre este repositório

Repositório criado para armazenar os exercícios, anotações e projetos práticos desenvolvidos durante o curso **[Java COMPLETO — Programação Orientada a Objetos + Projetos](https://www.udemy.com/course/java-curso-completo/)**, ministrado pelo **Prof. Dr. Nelio Alves** na plataforma Udemy.

O curso cobre Java do zero até o nível avançado, com forte base em **Orientação a Objetos**, diagramas UML, e projetos reais.

---

## Conteúdo do Curso

| # | Módulo |
|---|--------|
| 01 | Introdução e conceitos básicos |
| 02 | Estrutura sequencial, condicional e repetitiva |
| 03 | Introdução à Programação Orientada a Objetos (POO) |
| 04 | Construtores, `this`, sobrecarga e encapsulamento |
| 05 | Comportamento de memória, arrays e listas |
| 06 | Tópicos especiais em Java (Date, enumerações, composição) |
| 07 | Herança e polimorfismo |
| 08 | Tratamento de exceções |
| 09 | **Projeto: Jogo de Xadrez** |
| 10 | Interfaces |
| 11 | Generics, Set e Map |
| 12 | Programação funcional e expressões lambda |
| 13 | Acesso a banco de dados com JDBC |
| 14 | Interface gráfica com JavaFX |
| 15 | JPA / Hibernate |
| 16 | Spring Boot |
